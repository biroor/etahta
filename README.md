🎓 Biroor eThreadboard
A software solution specifically designed for smart boards, developed to provide equal opportunities in education.

✨ Features
User-Friendly: Improved adaptability for teachers and users.
Accessibility: Lightweight and fast operation on all types of hardware.
Free and Non-Profit: Completely free to support education.
🚀 Installation
To run the project on your local machine, follow these steps:

Store the clone: ​​git clone https://github.com/biroor/etahta.git
Enter the necessary credentials: npm install
Start the application: npm start
